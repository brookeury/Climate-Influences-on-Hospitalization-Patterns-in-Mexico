# Climate-Influences-on-Hospitalization-Patterns-in-Mexico

This directory has the code needed to replicate the analysis in "Climate Influences on Hospitalization Patterns in Mexico: Evidence from 30 Million Records" by Ury et al.

To replicate the results and run the scripts, the following data is required:

The Subsistema Automatizado de Egresos Hospitalarios (SAEH) dataset, published by the Sistema Nacional de Informaci√≥n en Salud (SINAIS) of the Mexican Department of Health. The data can be found here: http://www.dgis.salud.gob.mx/contenidos/sinais/s_saeh.html

Temperature data is available from the North American Regional Reanalysis (NARR) model published by the National Centers for Environmental Prediction (NCEP): https://psl.noaa.gov/data/gridded/data.narr.html

Precipitation data is available from the Climate Hazards Group InfraRed Precipitation with Station data (CHIRPS) model published by the Climate Hazard Center at the University of California, Santa Barbara: https://www.chc.ucsb.edu/data/chirps
